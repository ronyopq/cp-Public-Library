# Final Setup

After the first production deploy:

1. Open the deployed Worker URL.
2. Complete the bootstrap flow and create the first `super_admin`.
3. Confirm `Asia/Dhaka` branding/settings values are correct.
4. Upload a sample member photo and book cover.
5. Run a sample issue/return/payment/export flow.
6. Enable public modules only after verification.
